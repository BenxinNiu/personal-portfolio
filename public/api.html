Image Search Abstraction Layer API (2017)

User can get the image URLs, alt text and page urls for a set of images relating to a given search string.

User can paginate through the responses by adding a ?offset=2 parameter to the URL.

User can get a list of the most recently submitted search strings..

The API is powered by Node.js, MongoDB, and bing search API

File Metadata Microservice API (2017)

User can submit a FormData object that includes a file upload.

When user submit something, user will receive the file information within the JSON response

The API is powered by Node.js, and multer module

URL Shortener Microservice API (2017)

User can pass a URL as a parameter and will receive a shortened URL in the JSON response.

When user visit that shortened URL, it will redirect to the original link.

The API is powered by Node.js, and MongoDB

Timestamp Microservice API (2017)

User can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date

If it does, it returns both the Unix timestamp and the natural language form of that date.

The API is powered by Node.js
